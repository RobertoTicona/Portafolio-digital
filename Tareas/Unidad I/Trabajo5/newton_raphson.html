Descripci√≥n

El m√©todo de Newton-Raphson es un procedimiento iterativo utilizado para encontrar aproximaciones sucesivas de las ra√≠ces reales de una funci√≥n continua y derivable. Su fundamento radica en el desarrollo del polinomio de Taylor de primer orden alrededor de un punto inicial x‚ÇÄ, lo cual permite estimar el punto donde la funci√≥n corta el eje x mediante la siguiente f√≥rmula recursiva: 

x‚Çô‚Çä‚ÇÅ = x‚Çô - f(x‚Çô) / f'(x‚Çô)

En cada iteraci√≥n, el m√©todo utiliza la pendiente de la tangente en el punto x‚Çô para aproximarse progresivamente hacia la ra√≠z. Si la funci√≥n es suficientemente suave y el punto inicial se elige adecuadamente, la convergencia suele ser r√°pida y cuadr√°tica.

Aplicaci√≥n del m√©todo en Python

El objetivo fue desarrollar en el lenguaje de programaci√≥n Python un programa que permita al usuario ingresar una funci√≥n f(x) y graficarla en un intervalo definido. Esta etapa inicial tiene como prop√≥sito ayudar al usuario a identificar visualmente las posibles ra√≠ces y decidir si desea aplicar el m√©todo de Newton-Raphson.

En caso afirmativo, el programa solicita un valor inicial x‚ÇÅ basado en la observaci√≥n de la gr√°fica y ejecuta el algoritmo iterativo de Newton-Raphson hasta que la diferencia entre dos aproximaciones sucesivas sea menor a una tolerancia predefinida. Finalmente, el programa muestra en pantalla la ra√≠z aproximada encontrada y el n√∫mero de iteraciones necesarias para alcanzarla.

Este enfoque combina la interpretaci√≥n gr√°fica con el an√°lisis num√©rico, promoviendo una comprensi√≥n m√°s completa del comportamiento de la funci√≥n y de la eficacia del m√©todo iterativo.

      import numpy as np
      import matplotlib.pyplot as plt

      # === 1. Ingreso de la funci√≥n ===
      func_str = input("Ingrese la funci√≥n f(x): ")  # Ejemplo: x**3 - x - 1

      # Definimos la funci√≥n y su derivada (usando derivada num√©rica)
      def f(x):
          return eval(func_str)

      def f_prime(x, h=1e-6):  # derivada num√©rica
          return (f(x + h) - f(x - h)) / (2 * h)

      # === 2. Graficar la funci√≥n ===
      xmin = float(input("Ingrese el valor m√≠nimo de x: "))
      xmax = float(input("Ingrese el valor m√°ximo de x: "))

      x = np.linspace(xmin, xmax, 400)
      y = f(x)

      plt.figure(figsize=(8, 5))
      plt.plot(x, y, label=f"f(x) = {func_str}", color='blue')
      plt.axhline(0, color='black', linestyle='--')
      plt.axvline(0, color='black', linestyle='--')
      plt.title("Gr√°fico de la funci√≥n ingresada")
      plt.xlabel("x")
      plt.ylabel("f(x)")
      plt.legend()
      plt.grid(True)
      plt.show()

      # === 3. Pregunta si desea aplicar Newton-Raphson ===
      op = input("¬øDesea encontrar una ra√≠z con el m√©todo de Newton-Raphson? (s/n): ").lower()

      if op == "s":
          # === 4. Ingreso de punto inicial ===
          x0 = float(input("Basado en la gr√°fica, ingrese el valor inicial x1: "))

          # Par√°metros del m√©todo
          tol = 1e-6
          max_iter = 100

          # Iteraciones
          for i in range(1, max_iter + 1):
              fx = f(x0)
              fpx = f_prime(x0)

              if fpx == 0:
                  print(f"La derivada es cero en x = {x0}. El m√©todo no puede continuar.")
                  break

              x1 = x0 - fx / fpx

              # Verificar convergencia
              if abs(x1 - x0) < tol:
                  print(f"\n‚úÖ Ra√≠z aproximada encontrada: {x1:.6f}")
                  print(f"Iteraciones realizadas: {i}")
                  break

              x0 = x1
          else:
              print("\n‚ö†Ô∏è No se alcanz√≥ la convergencia despu√©s de", max_iter, "iteraciones.")

      else:
          print("No se aplic√≥ el m√©todo de Newton-Raphson.")
