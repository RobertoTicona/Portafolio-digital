Actividad del curso de Programaci√≥n Num√©rica

Descripci√≥n

El m√©todo del Punto Fijo es una t√©cnica iterativa utilizada para encontrar soluciones aproximadas de ecuaciones no lineales de la forma f(x) = 0. Se basa en reescribir la ecuaci√≥n en una forma equivalente: 

x = g(x)

En este enfoque, la ra√≠z buscada corresponde al punto donde g(x) y la recta y = x se intersectan. A partir de un valor inicial x‚ÇÄ, se genera una sucesi√≥n de aproximaciones mediante: 

x‚Çô‚Çä‚ÇÅ = g(x‚Çô)

El proceso contin√∫a hasta que la diferencia entre dos iteraciones consecutivas sea menor que una tolerancia predefinida, indicando que se ha alcanzado una convergencia aceptable.

Para que el m√©todo converja, es necesario que g(x) sea continua y diferenciable en el intervalo considerado, y que se cumpla la condici√≥n |g'(x)| < 1. En caso contrario, el m√©todo puede divergir o entrar en oscilaci√≥n.

Antes de aplicar el m√©todo, se recomienda representar gr√°ficamente tanto la funci√≥n f(x) como su forma iterativa g(x) en un intervalo adecuado. Esta visualizaci√≥n ayuda a identificar las zonas de intersecci√≥n y seleccionar un valor inicial x‚ÇÄ apropiado.

Entre sus ventajas destacan su simplicidad conceptual y la facilidad de implementaci√≥n. Sin embargo, requiere elegir cuidadosamente la forma de g(x) y el valor inicial para asegurar la convergencia.

Aplicaci√≥n del m√©todo en Python

Se implement√≥ un programa en Python que permite ingresar una funci√≥n f(x) y su forma iterativa g(x), graficar la funci√≥n y aplicar el m√©todo de Punto Fijo de manera interactiva.

El programa eval√∫a de forma sucesiva x‚Çô‚Çä‚ÇÅ = g(x‚Çô) hasta que el error entre iteraciones sea menor que una tolerancia establecida. Tambi√©n muestra los valores obtenidos en cada paso, facilitando el an√°lisis de la convergencia.

Esta pr√°ctica permite comprender el comportamiento del m√©todo y observar c√≥mo influye la elecci√≥n de la funci√≥n iterativa g(x) en la estabilidad del proceso.

C√≥digo en Python


      import numpy as np
      import matplotlib.pyplot as plt

      # === 1. Ingreso de la funci√≥n y su forma iterativa g(x) ===
      print("=== M√âTODO DEL PUNTO FIJO ===")
      print("Recuerde que f(x) = 0 se reescribe como x = g(x)")
      print("Ejemplo: Si f(x) = cos(x) - x, entonces g(x) = cos(x)\n")

      func_str = input("Ingrese la funci√≥n original f(x): ")   # Ejemplo: np.cos(x) - x
      g_str = input("Ingrese la funci√≥n iterativa g(x): ")     # Ejemplo: np.cos(x)

      # Definici√≥n de funciones
      def f(x):
          return eval(func_str, {"np": np, "x": x})

      def g(x):
          return eval(g_str, {"np": np, "x": x})

      # === 2. Graficar la funci√≥n f(x) ===
      xmin = float(input("Ingrese el valor m√≠nimo de x: "))
      xmax = float(input("Ingrese el valor m√°ximo de x: "))

      x = np.linspace(xmin, xmax, 400)
      y = f(x)

      plt.figure(figsize=(8, 5))
      plt.plot(x, y, label=f"f(x) = {func_str}", color='blue')
      plt.axhline(0, color='black', linestyle='--')
      plt.axvline(0, color='black', linestyle='--')
      plt.title("Gr√°fico de la funci√≥n ingresada")
      plt.xlabel("x")
      plt.ylabel("f(x)")
      plt.legend()
      plt.grid(True)
      plt.show()

      # === 3. Pregunta si desea aplicar el m√©todo de punto fijo ===
      op = input("¬øDesea aplicar el m√©todo de Punto Fijo? (s/n): ").lower()

      if op == "s":
          # === 4. Ingreso de valor inicial ===
          x0 = float(input("Ingrese el valor inicial x0: "))

          tol = 1e-6
          max_iter = 100

          print("\nIteraci√≥n |     x0      |     g(x0)     |     f(x0)     |   Error")
          print("---------------------------------------------------------------")

          for i in range(1, max_iter + 1):
              x1 = g(x0)
              error = abs(x1 - x0)

              print(f"{i:9d} | {x0:10.6f} | {x1:12.6f} | {f(x1):12.6f} | {error:10.6f}")

              if error < tol:
                  print(f"\n‚úÖ Ra√≠z aproximada encontrada: {x1:.6f}")
                  print(f"Iteraciones realizadas: {i}")
                  break

              x0 = x1
          else:
              print("\n‚ö†Ô∏è No se alcanz√≥ la convergencia despu√©s de", max_iter, "iteraciones.")
      else:
          print("No se aplic√≥ el m√©todo de Punto Fijo.")
