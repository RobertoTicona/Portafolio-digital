Descripci√≥n

El m√©todo de la Secante es un procedimiento num√©rico iterativo empleado para aproximar las ra√≠ces reales de una funci√≥n continua f(x). A diferencia del m√©todo de Newton-Raphson, este m√©todo no requiere el c√°lculo de la derivada anal√≠tica, ya que la pendiente se aproxima usando dos puntos consecutivos cercanos a la ra√≠z.

La idea central consiste en reemplazar la tangente utilizada en el m√©todo de Newton por una secante, es decir, la recta que pasa por los puntos (xn-1, f(xn-1)) y (xn, f(xn)). La intersecci√≥n de esta recta con el eje x proporciona una nueva aproximaci√≥n de la ra√≠z, calculada mediante la siguiente f√≥rmula: 

x‚Çô‚Çä‚ÇÅ = x‚Çô - f(x‚Çô) * (x‚Çô - x‚Çô‚Çã‚ÇÅ) / (f(x‚Çô) - f(x‚Çô‚Çã‚ÇÅ))

Este m√©todo combina la simplicidad del m√©todo de bisecci√≥n con una velocidad de convergencia superior, cercana a la del m√©todo de Newton-Raphson, siempre que los valores iniciales sean adecuados.

Antes de aplicar el m√©todo, es recomendable representar gr√°ficamente la funci√≥n f(x) en un intervalo apropiado. Esta visualizaci√≥n ayuda a seleccionar los puntos iniciales x‚ÇÄ y x‚ÇÅ pr√≥ximos a una ra√≠z, lo cual mejora la estabilidad y rapidez de convergencia del m√©todo.

Entre sus principales ventajas destacan su eficiencia y el hecho de que no requiere derivadas. Sin embargo, su √©xito depende de la elecci√≥n adecuada de los puntos iniciales y de la continuidad de la funci√≥n, ya que, en caso contrario, el m√©todo podr√≠a divergir.

Aplicaci√≥n del m√©todo en Python

Se implement√≥ un programa en Python que permite al usuario ingresar una funci√≥n f(x) y graficarla en un intervalo definido. Esta primera etapa facilita la identificaci√≥n visual de las posibles ra√≠ces y la selecci√≥n de los valores iniciales x‚ÇÄ y x‚ÇÅ.

Posteriormente, el programa aplica el m√©todo de la Secante de manera iterativa hasta alcanzar una tolerancia establecida o un n√∫mero m√°ximo de iteraciones. En cada paso, se muestran los valores de x‚ÇÄ, x‚ÇÅ, f(x‚ÇÄ), f(x‚ÇÅ), la nueva aproximaci√≥n x‚ÇÇ y el error absoluto.

Este enfoque integra la interpretaci√≥n gr√°fica con el c√°lculo num√©rico, promoviendo una comprensi√≥n m√°s profunda del comportamiento y la convergencia del m√©todo.

C√≥digo en Python


      import numpy as np
      import matplotlib.pyplot as plt

      # === 1. Ingreso de la funci√≥n ===
      func_str = input("Ingrese la funci√≥n f(x): ")  # Ejemplo: x**3 - x - 1

      # Definir la funci√≥n
      def f(x):
          return eval(func_str, {"np": np, "x": x})

      # === 2. Graficar la funci√≥n ===
      xmin = float(input("Ingrese el valor m√≠nimo de x: "))
      xmax = float(input("Ingrese el valor m√°ximo de x: "))

      x = np.linspace(xmin, xmax, 400)
      y = f(x)

      plt.figure(figsize=(8, 5))
      plt.plot(x, y, label=f"f(x) = {func_str}", color='blue')
      plt.axhline(0, color='black', linestyle='--')
      plt.axvline(0, color='black', linestyle='--')
      plt.title("Gr√°fico de la funci√≥n ingresada")
      plt.xlabel("x")
      plt.ylabel("f(x)")
      plt.legend()
      plt.grid(True)
      plt.show()

      # === 3. Pregunta si desea aplicar el m√©todo de la Secante ===
      op = input("¬øDesea encontrar una ra√≠z con el m√©todo de la Secante? (s/n): ").lower()

      if op == "s":
          # === 4. Ingreso de puntos iniciales ===
          x0 = float(input("Ingrese el primer valor inicial x0: "))
          x1 = float(input("Ingrese el segundo valor inicial x1: "))

          tol = 1e-6
          max_iter = 100

          print("\nIteraci√≥n |     x0      |      x1      |     f(x0)     |     f(x1)     |     x2      |   Error")
          print("----------------------------------------------------------------------")

          for i in range(1, max_iter + 1):
              f0 = f(x0)
              f1 = f(x1)

              if f1 - f0 == 0:
                  print(f"\n‚ö†Ô∏è Divisi√≥n por cero en la iteraci√≥n {i}. El m√©todo no puede continuar.")
                  break

              x2 = x1 - f1 * (x1 - x0) / (f1 - f0)
              error = abs(x2 - x1)

              print(f"{i:9d} | {x0:10.6f} | {x1:10.6f} | {f0:12.6f} | {f1:12.6f} | {x2:10.6f} | {error:10.6f}")

              if error < tol:
                  print(f"\n‚úÖ Ra√≠z aproximada encontrada: {x2:.6f}")
                  print(f"Iteraciones realizadas: {i}")
                  break

              x0, x1 = x1, x2

          else:
              print("\n‚ö†Ô∏è No se alcanz√≥ la convergencia despu√©s de", max_iter, "iteraciones.")
      else:
          print("No se aplic√≥ el m√©todo de la Secante.")
